Barcode Registry — Overview

Slides (HTML) — use ← → or buttons

A tiny local DApp for registering & verifying barcoded products. Built with:

  • Solidity contract: BarcodeRegistry.sol
  • Hardhat for local node & deployment
  • Vanilla front-end using ethers.js + MetaMask

Architecture

Browser ↔ MetaMask ↔ Hardhat RPC ↔ Contract

Interactive diagram (embedded):

Contract — key functions

BarcodeRegistry.sol
  • addProducer(address, string) — owner-only: allow producer addresses
  • removeProducer(address) — owner-only: revoke permission
  • registerProduct(barcode, name, producerAddr) — onlyProducerOrOwner
  • verifyProduct(barcode) — view: returns (exists, name, producer)

Front-end

presentation/index.html + presentation/app.js

Features:

  • Connect MetaMask; request switch to local Hardhat chain (31337)
  • Register products (owner or allowed producers)
  • Add / remove producers (owner UI)
  • Query past events to populate product & producer lists

How to run locally

Quick commands
  1. Install dependencies (npm): 
    npm install
  2. Run Hardhat node: 
    npx hardhat node
  3. In another terminal deploy: 
    npx hardhat run --network localhost scripts/deploy.js
  4. Serve the front-end and open in browser (or open file): 
    cd presentation && python3 -m http.server 8000
  5. Open http://127.0.0.1:8000 and connect MetaMask (import owner key if needed)
